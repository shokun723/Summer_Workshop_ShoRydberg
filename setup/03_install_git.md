## 3. Install Git

Git is required to clone repositories and save your work.

---

1. Download Git

Go to:

https://git-scm.com/

Click:

Download for Windows

---

2. Run the installer

Open the downloaded file.

Click:

Next

Continue clicking Next through most steps.

---

3. Important settings

Make sure the following option is selected:

Git from the command line and also from 3rd-party software

---

4. Finish installation

Click:

Install

Then click:

Finish

---

5. Restart VS Code

Close VS Code and open it again.

---

6. Verify installation

Open VS Code terminal and run:

git --version

You should see something like:

git version 2.xx.x

---

7. Confirm Git is working

Press:

Ctrl + Shift + P

Type:

Git: Clone

You should now see the option available.
